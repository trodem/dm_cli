# Vim

## Basics
- Open file: `vim file.txt`
- Save: `:w`
- Quit: `:q`
- Save and quit: `:wq`
- Force quit: `:q!`

## Modes
- Normal: default mode
- Insert: `i`, `a`, `o`
- Visual: `v`, `V`, `Ctrl+v`
- Command: `:`

## Movement
- Word: `w`, `b`, `e`
- Line: `0`, `^`, `$`
- File: `gg`, `G`
- Page: `Ctrl+f`, `Ctrl+b`
