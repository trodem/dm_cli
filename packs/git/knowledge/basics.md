# Git Basics

## Common
- Status: `git status`
- Stage all: `git add .`
- Commit: `git commit -m "msg"`
- Log: `git log --oneline --graph --decorate`

## Branching
- List: `git branch`
- Create: `git checkout -b feature/x`
- Switch: `git checkout main`

## Remote
- Add: `git remote add origin <url>`
- Pull: `git pull`
- Push: `git push -u origin main`
